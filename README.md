# The Cognitive Revolution Explorer

An interactive, visual exploration of how AI is transforming industries — built as a "Where's Waldo"-style illustrated collage.

## Overview

This project visualizes insights from [Paul DiMaggio's Substack](https://pauldimaggio.substack.com) about the AI-driven cognitive revolution across 11 industries:

- IT Services & MSPs
- Software Development
- Consulting
- Finance
- Healthcare
- Legal
- Education
- Labor Market
- Music
- Video Production
- Publishing

## Features

- **Interactive Canvas**: Pan, zoom, and explore a grid of illustrated industry scenes
- **Detail Zoom**: Click any industry to expand into a detailed view with key insights
- **Responsive Design**: Optimized for both desktop (4-column grid) and mobile (2-column grid)
- **Custom Illustrations**: 43 AI-generated illustrations in a clean animated style

## Tech Stack

- Vanilla HTML/CSS/JavaScript
- Canvas-based pan/zoom with lerp animations
- No build tools or frameworks required

## Getting Started

Simply open `index.html` in a browser, or serve the directory with any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .
```
